# To-Do List React App

A simple To-Do List web application built using React.
This project is created to practice React basics, Git, and GitHub.

## Features
- Add tasks
- View tasks
- Delete tasks

## Technologies Used
- React
- JavaScript
- HTML, CSS
- Git & GitHub

## How to Run
1. Clone the repository
2. Install dependencies:
   npm install
3. Start the app:
   npm start
